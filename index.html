Abstract

The National High Magnetic Field Laboratory performs an experiment using Nuclear Magnetic Resonance (NMR) probes. The experiment uses radio frequency (RF) pulses and an oscilloscope to observe the incoming and reflecting power. The purpose of the oscilloscope is to assist the user in adjusting, or tuning, the probes to prevent damage within them and the experiment. The goal of the project is to use a Software Defined Radio (SDR) to replace the oscilloscope by having it mimic its functionality to observe the RF pulses. By using two Adalm Pluto SDRs and their receiver ports, the incoming and reflecting RF pulses data can be used on a computer using software. Using the software, the ratio of the RF pulses is calculated and displayed on a graph for the user to observe. The SDRs are synchronized physically as well as through the software to help line up the data with the experiment. This allows them to effectively tune the probes to prevent damage within them. We were able to accurately measure, record, and display the RF pulses from the experiment. With the success of this project, the Magnetic Lab no longer needs to spend an excessive amount of money on oscilloscopes to monitor these NMR experiments. This allows them to use the oscilloscopes for other purposes instead of the simple purpose of a quick observation.
